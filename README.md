# Gulp-Babel-nextcss-boilerplate

### Boilerplate contains:
* Gulp
* Babel: Transpiling ES2015 => ES5
* CSSnext: Next generation of CSS
* CSSComb: to keep css files consistant
* ESLint: to keep js files consistent
* Pre-commit

### Getting started
* npm i
* npm run dev

### Other scripts
* npm run build
* npm run cleancss
* npm run lintjs
